# vine UI
> Lightweight UI base on vue 3.x.
* Minified: 69.2 KB
* Gzip: 15.7 KB
## Components
* Button
* Checkbox
* Dropdown
* Flex
* Flyover
* Input
* Loading
* Modal
* Popover
* Radio
* Select
* Tab
* Tooltip
## Install
```sh
npm i vine-ui
```
## Usage
```
import {
    VuiCheckbox,
    VuiInput
} from "vine-ui";

export default {
    components: {
        VuiCheckbox,
        VuiInput
    }
}
```
see [demo.vue](public/src/demo.vue)
